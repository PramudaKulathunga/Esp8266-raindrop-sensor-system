--telegram_raindrop_sensor--
* This is esp8266 base rain checking system with telegram bot.

--How to make telegram bot--
* First you visit BotFather in telegram.
* There are lot of option for you. If you haven't create telegram bot, press "/newbot".
* Then they will ask name for bot. Fill that as you wish. 
* After they will ask username for your bot. It should be like "example_bot".
* After they prepare bot for you & they will give access token.
* Copy that token and paste it in your code.
